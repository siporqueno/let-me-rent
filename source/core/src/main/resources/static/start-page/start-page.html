Добро пожаловать на сайт для поиска и размещения предложений по аренде инструментов
Ознакомьтесь с правилами пользования сайтом:
  1. Без регистрации/авторизации доступен следующий функционал:
    • Просмотр списка предложенных для аренды инструментов, в том числе, подробной информации о интересующих инструментах. В общей таблице доступна фильтрация и сортировка элементов
    • Возможность положить инструменты в корзину (для этого потребуется в полях фильтрации указать период времени, в рамках которого вы планируете арендовать инструмент)
    • Возможность добавить инструменты в "Избранное"
  2. Регистрация/авторизация необходима для доступа к следующим возможностям сайта:
    • Добавление своего инструмента для предложения в аренду на портале
    • Доступ к личнопу кабинету (в Личном кабинете размещена подробная информация об арендованных пользователем инструментах и своих инструментах, выставленных для сдачи в аренду и весь сопутствующий функционал)
    • Оформление аренды инструментов
    • Отправка отзывов об арендованных инструментах и об арендаторах, с которыми довелось взаимодействовать


Cтандартные условия договора аренды
  1. Цена аренды на сайте указана за сутки
  2. День приема инструмента арендатором включается в расчет стоимости аренды, если прием осуществлен до 17:00 МСК включительно
  3. День возврата инструмента арендатором включается в расчет стоимости аренды, если возврат осуществлен после 17:00 МСК
  4. Размер залога, установленный на сайте, соответствует стоимости инструмента с учетом текущего износа
  5. Залог передается в момент приема инстурмента арендатором и возвращается в момент возврата инструмента владельцу
  6. В случае поломки инструмента по вине арендатора:
    • Стоимость ремонта инсрумента удерживается из залога
    • Стоимость аренды не пересчитывается
  7. В случае поломки инструмента по причине износа:
    • Ремонт инструмента осуществляет владелец за свой счет
    • Стоимость аренды пересчитывается исходя из фактического количества дней пользования инструментом
